# Lessons Learned with FT-991A

1. VARAC Sucks for linux. Its plain and simple.  Should be a native linux version.  Prove me wrong. 
2. VARAC does not work well with Winlink, running under same localhost port as VARAC that is listed in VARA HF.  Both applications cannot connect to VARA HF on 8300 at same time.  Need to switch off one of them. 
3. RIGCTLD does work to key radio but if 2 above is not met. Radio will not key and transmit data.  This makes user think something is wrong with radio configuration.  Bad things happen.
